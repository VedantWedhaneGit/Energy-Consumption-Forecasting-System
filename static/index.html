Electrical Energy Forecasting System

Electricity forecasting involves predicting future electricity demand and supply to optimize energy production and distribution. Accurate forecasting is essential for balancing supply and demand, reducing operational costs, and ensuring the reliability of the power grid.

Our project aims to develop a comprehensive forecasting system across multiple sectors. We utilize data sources from various sectors to ensure accurate and reliable predictions. Our process includes cleaning and preprocessing the data to ensure its quality and suitability for modeling. We use Python with Flask to power our backend logic and handle API interactions on the front end. The forecasting models we employ include:

  • ARIMA
  • ANN
  • SARIMA
  • ETS
  • Prophet
  • LSTM
  • Hybrid models

These models are used to make precise predictions, and we use Flask for prediction and visualization tasks.

    About the Dataset

  • The dataset we chose for the project is energy consumption.
  • It contains two columns: datetime (ds) and consumption (y).
  • Datetime is in hours.



Forecasting Models
Model Description Key Points
ARIMA Autoregressive Integrated Moving Average model
  • Statistical time series forecasting method
  • Suitable for univariate time series data
  • Requires stationarity in the time series
  • Considers autoregression, differencing, and moving average components
  • Popular choice for short-term forecasting
ANN Artificial Neural Network model
  • Deep learning model inspired by the human brain
  • Capable of learning complex patterns in data
  • Requires large amounts of data for training
  • Used for both regression and classification tasks
  • Flexible architecture with multiple layers
SARIMA Seasonal Autoregressive Integrated Moving Average model
  • Extension of ARIMA for seasonal time series data
  • Handles seasonality in addition to trend and noise
  • Requires identification of seasonal periods
  • Suitable for data with clear seasonal patterns
  • Used for medium to long-term forecasting
ETS Error-Trend-Seasonality model
  • Exponential smoothing method for time series forecasting
  • Models trend and seasonality components separately
  • Applicable to data with no clear underlying model
  • Requires estimation of smoothing parameters
  • Useful for short-term forecasting and noise reduction
Prophet Prophet model by Facebook
  • Time series forecasting tool developed by Facebook
  • Handles seasonality, holidays, and outliers automatically
  • Easy-to-use interface with intuitive parameters
  • Designed for analysts with little or no expertise in forecasting
  • Uses additive model with piecewise linear or logistic growth
LSTM Long Short-Term Memory model
  • Deep learning model for sequence prediction tasks
  • Capable of learning long-term dependencies
  • Uses gated recurrent units to overcome vanishing gradient problem
  • Effective for time series data with complex patterns
  • Requires substantial computational resources for training
Hybrid models Combination of multiple forecasting models
  • Blend different forecasting techniques for improved accuracy
  • Combines strengths of individual models
  • Requires careful selection of constituent models
  • Can handle various types of data and forecasting horizons
  • Commonly used in industry for robust predictions
